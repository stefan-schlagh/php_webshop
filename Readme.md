# php Webshop

## install dependencies

````shell
sh installDepemdencies.sh
````
does not work on Windows

## start

* install docker and docker-compose
* execute the following command:

````shell
docker-compose up -d
````

### phpmyadmin

can be reached at localhost:5000
* user: root
* password: secret

### webshop

can be reached at localhost:8080
